About this project

What is this?

This is my-project — describe your project here. Built with Refract, which gives you a REST API, CLI, MCP tools, and a Web UI from a single Python function.

How to add a function

Create a function in my_project/core/, decorate it with @register_function, and it automatically appears in the API, CLI, and Web UI — no extra wiring needed.

How to add a UI element

Create a Web Component in my_project/web/elements/<name>/index.js extending AutoFunctionController from the Refract SDK (/refract/controller.js). Add it to a view and import the script — that's it.
